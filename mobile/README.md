# Fredy Unlimited AI - Mobile App

## 🚀 Quick Start

### Installation

```bash
npm install
npx expo install expo-image-picker
npx expo install @react-native-async-storage/async-storage
cp .env.example .env
```

### Run

```bash
npm start
npm run android
```

## 📱 Features

- Photo Upload
- Document Analysis
- Letter Generation
- User Authentication

## 🏗️ Build APK

```bash
npm install -g eas-cli
eas build:configure --platform android
npm run build:android
```
